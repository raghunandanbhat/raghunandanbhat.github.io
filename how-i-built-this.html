How I built this simple Website

I love simple things. It's not that I don't like websites with nice aesthetics, I like the fact that this website works. It is simple, fast and responsive.


I wanted to build my own portfolio website for a long time. But I kept on postponing it, finding some or other excuses. The fact that I had to host it somewhere which was free and the thing with free services on internet is that they don't last long. They run perfectly for few years and suddenly one fine day they close shop.


Then GitHub came up with github.io sites. This time the excuse was building aesthetics. It looks nice and all but the truth is building such webiste is hard and time consuming. And that's when I found Andrej Karpathy's webiste.


It was simple and it just worked. As I scrolled to the end, I found (please open this website and thank me later) exactly what I was looking for. As I was going through the website, I imagined Samuel L Jackson reading it to me. Then I found 1 MB Club and 512 KB Club.


And that was it. I found my inspiration. I scrolled bunch of those websites. Took whatever I liked, wrote some HTML and CSS and it was ready - My Website


Feel free to take whatever you want from this webiste to build yours. I will try my best to keep updating it but remember - Free things on the internet do not last long.
